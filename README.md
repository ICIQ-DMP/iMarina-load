# iMarina-load
Scripts to obtain A3 data, transform it into iMarina load format, and upload it to iMarina server using SFTP


## Usage

### Install python
In my Ubuntu is:
```shell
sudo apt install python3.12-venv gcc build-essential -y
```



### Initialize venv
```shell
cd iMarina
python3 -m venv venv
./venv/bin/pip install -r requirements.txt 
```
